Firstly, you need to install the Crystal Reports plugin on your computer in order to open the project. Otherwise, the program will not launch.

LINK: https://origin-az.softwaredownloads.sap.com/public/file/0020000001649932022

The reporting section within the program will not function after you download it. Since the reports are created with information from a local database connection, they must be recreated after database integration. Setup files are available in the program files, but first, you should customize the ConnectionString according to your needs and then rebuild the Setup.

This project involves an automation system for a ticket sales office. The program aims to facilitate the sale of bus, plane, and train tickets on the same platform, along with providing administrative tools. Additionally, the program has the capability to generate boarding passes in PDF format based on specific information after ticket sales.
